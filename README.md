### Safety Documentation

This repository contains important safety documentation:
- SDS (Safety Data Sheets) for all materials used
- BYU Engineering Lab Safety Plan
- Safety and Health Risk Assessment matrices
- [SOP Template](./byu-engineering-sop-template.md) for creating new SOPs

### Getting Started

Before performing any procedure:
1. Read the relevant SOP document in full
2. Review all associated Safety Data Sheets (SDS)
3. Complete required training as specified in the SOP
4. Discuss with your supervisor before first-time execution
